# [GamesCollector](https://gamescollector.netlify.app/)

GamesCollector is a passion-driven project built for gamers, by gamers. My goal is to create a central hub where users can discover games, track new releases, build personalized game collections, rate games, and share their experiences.<br><br>

## Tech Stack

This project is built using the following technologies:

- **React** - For building the user interface
- **TypeScript** - Ensuring type safety and better developer experience
- **Next.js** - Providing server-side rendering and optimized performance
- **PandaCSS** - Used for styling and maintaining a consistent design system
- **Supabase** - Backend-as-a-service for authentication and real-time capabilities
- **PostgreSQL** - Database for storing game-related data and user interactions<br><br>

## Purpose

GamesCollector is a non-commercial project created purely for fun. It is an evolving platform aimed at bringing gamers together and providing valuable game-related information without any financial objectives.<br><br>

## Status

The project is currently in its early development stages. Any feedback, contributions and ideas are welcomed.<br><br>
